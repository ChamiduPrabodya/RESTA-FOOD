Backend Test Evidence Pack

This folder contains text-based test cases and evidence notes for the backend work in this project.

Files:
- auth_user_test_cases.txt
- checkout_loyalty_test_cases.txt
- payhere_test_cases.txt
- database_test_cases.txt
- executed_evidence.txt

Scope covered:
- Authentication and user profile endpoints
- Cart, checkout, orders, purchases, and loyalty flows
- PayHere configuration and notification handling
- MongoDB connection, collections, and index sync

How to use:
1. Run the relevant feature manually.
2. Add screenshot filenames or terminal output below each test case in your report if needed.
3. Use executed_evidence.txt as proof of the checks already completed in this workspace.
